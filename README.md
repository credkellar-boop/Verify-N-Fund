# Verify-N-Fund 🏦🔍

**Verify-N-Fund** is an AI-powered automation blueprint designed for currency exchanges, check-cashing services, and financial institutions. By combining automated verification databases with a highly specialized Gemini AI Agent, this tool drastically cuts down the time compliance officers spend manually reviewing checks and looking up internal protocols.

## ✨ Key Features
* **Automated Risk Assessment:** Speeds up verification of MICR lines, routing numbers, and ID matches.
* **Guarded Gemini Expert:** An in-app AI assistant programmed *strictly* to answer questions regarding the verification process, company guidelines, and check legitimacy. 
* **Time-Saving Workflows:** Designed to handle high-volume, repetitive teller verification inquiries so your team can focus on complex edge cases.

## 🚀 Guardrail Enforcement
To prevent misuse or distractions, the integrated Gemini model uses strict system instructions. It will **only** respond to queries regarding check verification, fraud patterns, or project mechanics. Any off-topic queries (e.g., writing general code, answering trivia, general chat) will be politely blocked.
